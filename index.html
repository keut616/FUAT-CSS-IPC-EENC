  1. ເປີດໄຟລ໌ IMAM OPD DATASET.xlsx ຂອງທ່ານໃນ Excel.
  2. ໄປທີ່ File > Save As ແລ້ວເລືອກປະເພດໄຟລ໌ເປັນ CSV (Comma delimited) (*.csv).
  3. ໄປທີ່ໂຟນເດີທີ່ທ່ານ Save ໄຟລ໌ນັ້ນໄວ້, ຄລິກຂວາໃສ່ໄຟລ໌ CSV ນັ້ນ > ເລືອກ Open with > ເລືອກ Notepad.
  4. ຫຼັງຈາກນັ້ນໃຫ້ກົດ Ctrl+A (ເພື່ອຄອບງຳທັງໝົດ) ແລະ Ctrl+C (ເພື່ອກັອບປີ້).
  5. ກັບມາທີ່ໂຄດໃນໜ້າຈໍເບື້ອງຂວາມື, ເລື່ອນລົງໄປຫາແຖວທີ່ຂຽນວ່າ: <script type="text/plain" id="embedded-csv">
  6. ໃຫ້ທ່ານວາງຂໍ້ມູນລົງໄປແທນທີ່ຂໍ້ມູນຕົວຢ່າງເກົ່າ (ໃຫ້ຢູ່ລະຫວ່າງແທັກເປີດ ແລະ ແທັກປິດ </script>).
